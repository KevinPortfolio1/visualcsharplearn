Graph 資料結構（圖論）
-----------------------------------------------------------------------------------------------------------------
在資料結構與演算法中，「Graph（圖）」是一種非常重要的資料結構，用來表示節點（Nodes）與邊（Edges）之間的關係。

Graph 演算法的實際應用方向

應用領域	使用演算法			用途範例
社群網路	BFS、PageRank		好友推薦、找紅人
地圖導航	Dijkstra、A*		最短路徑、導航
推薦系統	Graph Embedding		商品推薦
搜尋引擎	PageRank			頁面排序
生物資訊	Pathfinding			基因互動分析
資安		異常偵測			攻擊路徑偵測
編譯器		Topological Sort	編譯順序、依賴處理
遊戲 AI		A*、Dijkstra		NPC 路徑規劃

機器學習算法 (Machine Learning Algorithms)
---------------------------------------------------------------------------------------------------------------------
🔷 1. 監督式學習（Supervised Learning）

🔍 目標：從已標記資料中學習「輸入 → 輸出」的對應關係

📌 常見演算法與應用：
演算法												實際應用場景
📈 線性回歸 Linear Regression						房價預測、銷售預測、經濟模型
🧠 羅吉斯迴歸 Logistic Regression					疾病預測（有沒有癌症）、客戶流失預測
🌳 決策樹 Decision Tree / 隨機森林 Random Forest		貸款核准、自動駕駛環境分類、信用風險評估
🎯 支援向量機 SVM									臉部辨識、郵件分類（垃圾 vs 正常）
🧠 神經網路 Neural Networks							影像辨識、語音識別、自然語言處理（NLP）
📚 K-近鄰 KNN										推薦系統、醫學診斷系統
🔍 XGBoost / LightGBM								資料比賽冠軍模型，常用於 Kaggle、金融建模

🔷 2. 非監督式學習（Unsupervised Learning）

🔍 目標：從無標籤資料中找出結構或分群規律

📌 常見演算法與應用：
演算法						實際應用場景
📦 K-Means					客戶分群、圖片分類、自動市場區隔
📐 主成分分析 PCA			特徵降維、視覺化、基因資料分析
🤖 自編碼器 Autoencoder		異常偵測（製造業）、資料降維、圖片重建
📚 層次分群 Hierarchical		社群關係網路分析、文檔分群
🌐 潛在語意分析 LSA			搜尋引擎、文章推薦、主題偵測（新聞、博客）

🔷 3. 強化學習（Reinforcement Learning）

🔍 目標：透過「試錯」，學會如何最大化獎勵

📌 常見演算法與應用：
演算法						實際應用場景
🕹️ Q-Learning				機器人導航、自動交易策略
🎮 Deep Q Network (DQN)		玩 Atari、控制遊戲角色
♟️ Policy Gradient			自走棋、圍棋、StarCraft 等策略型 AI
🚗 AlphaGo / AlphaZero		世界級棋類冠軍 AI，OpenAI Five、AlphaStar
🤖 Multi-agent RL			自動駕駛車協調、無人機群體控制

🔷 4. 生成模型（生成式 AI）

🔍 目標：從學到的分布中生成新資料（圖像、聲音、文字）

📌 常見演算法與應用：
演算法						實際應用場景
🎨 GAN（生成對抗網路）		AI 畫圖、以圖生圖、圖像風格轉換、深偽技術
🧠 Transformer				GPT（文字生成）、BERT（語意理解）、TTS 聲音合成
📷 VAE（變分自編碼器）		人臉生成、圖像重建、資料增強
🧑 Diffusion Models			Stable Diffusion、Midjourney、DALL·E
