🔧 一、開發與編輯工具

1. 程式碼編輯器（Code Editor）

	語法高亮、自動補全（IntelliSense）

	語法錯誤即時偵測（Roslyn 分析器）

	多重游標、格式化、重構建議

2. Solution Explorer（方案總管）

	管理整個專案（solution）與檔案

	支援拖拉、重新命名、加入/排除檔案

3. Toolbox（工具箱）

	拖拉式控制項（GUI 設計）

	主要用於 Windows Forms / WPF / Xamarin / .NET MAUI

4. Properties Window（屬性視窗）

	用來設定控制項、檔案、專案的屬性

	例如 GUI 控制項的 Text、Color、Event

🧪 二、除錯與測試工具

5. Debugger（除錯器）

	設置中斷點、逐步執行（Step Into, Step Over）

	即時檢查變數值（Watch, Locals, Autos）

	即時編輯程式碼（Edit and Continue）

6. Live Visual Tree / Live Property Explorer

	WPF 與 XAML 應用專用：即時查看 UI 元素結構與屬性

7. Test Explorer

	執行單元測試（Unit Tests）

	支援 MSTest、NUnit、xUnit

🔍 三、效能分析與診斷工具

8. Performance Profiler

	分析 CPU 使用率、記憶體使用、UI 回應時間

	有多種模式（如 .NET 追蹤器、記憶體 profiler）

9. Diagnostic Tools

	在除錯期間顯示應用執行狀態（CPU、事件、例外）

🌐 四、Web 與資料庫工具

10. SQL Server Object Explorer

	連接與操作資料庫（新增資料表、撰寫 SQL）

	支援 SQL Server、Azure、SQLite（需安裝擴充）

11. Server Explorer

	瀏覽伺服器、資料庫、Web 服務等資源

12. Connected Services

	整合 Azure、REST API、WCF、OpenAPI 等服務

一鍵產生用戶端代理程式碼

📦 五、專案與程式碼管理工具

13. NuGet 管理器

	套件管理工具（安裝/更新/移除 .NET 套件）

	對應 nuget.org 套件庫

14. Git 工具

	內建 Git 支援：版本控制、提交、合併、分支管理

	支援 GitHub、Azure DevOps

15. Task List / Error List

	顯示 TODO 記錄與建構錯誤訊息

	快速跳轉至相關程式碼

🛠️ 六、UI/UX 設計工具

16. XAML Designer

	設計 WPF / UWP / MAUI 的 UI

	支援雙向編輯（設計 + XAML ）

17. Resource Editor

	編輯 .resx 資源檔（文字、多語系、圖示）

18. Image Editor

	基本圖片處理（圖示、點陣圖）

🔁 七、跨平台與行動開發工具

19. .NET MAUI 工具

	開發跨平台（Android, iOS, Windows, macOS）應用

	取代 Xamarin.Forms，支援 XAML UI 設計

20. Android Device Manager / Emulator

	測試 Android 應用（Xamarin, MAUI）

	建立模擬器、部署 APK

🧩 八、擴充與自訂工具

21. Extensions（擴充功能）

	安裝額外功能：Python 支援、Unity 工具、Live Share 等

	常用：ReSharper, Visual Assist, CodeMaid, GitLens

22. Visual Studio Installer

	增加或移除開發工作負載（.NET, C++, Python, etc.）

🔄 九、部署與發行工具

23. Publish 工具

	發佈應用程式（單檔案、安裝包、MSIX、Docker）

	支援 Azure, IIS, FTP, 本機資料夾等方式

24. ClickOnce 發行精靈

	建立桌面應用的安裝包與自動更新功能
